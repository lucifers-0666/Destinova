### **2. Animation Performance**
```css
/* Use transform (GPU accelerated) */
transform: translate() scale();
/* Not: top, left, width (CPU intensive) */

/* Will-change hint */
.hero-bg-image {
  will-change: transform;
}
```

### **3. Reduced Elements on Mobile**
```css
@media (max-width: 768px) {
  .floating-shape {
```
